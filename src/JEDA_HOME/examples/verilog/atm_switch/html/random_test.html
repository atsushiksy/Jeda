13. Random Test

    A simple random test is provided in random_test.j.
    The test body is simply calling configuration functions in the tester class. The test can be configured to use various random distribution type for input and output speed control by using plus arguments.
     
      void squat_test() {
        printf( "Enter squat_test!\n" ) ;
       
        tester.set_config_reg() ;
       
        tester.set_rand_config() ;
       
        tester.start_receiver_checker_rnd() ;
       
        rand_test1() ;
       
       
        printf( "squat_test done!\n" ) ;

      }


    The actual test sends 100 cells each from all the ports concurrently.
     

      void rand_test1() {
        int i, j ;
       
        fork
          tester.send_rnd_cells( 0, 100 ) ;
          tester.send_rnd_cells( 1, 100 ) ;
          tester.send_rnd_cells( 2, 100 ) ;
          tester.send_rnd_cells( 3, 100 ) ;
        join
        tester.wait_done() ;
       
      }
